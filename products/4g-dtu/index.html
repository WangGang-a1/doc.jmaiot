4G-DTU 数据传输终端

4G LTE 转串口数据透传模块，支持全网通，RS232/RS485 接口，稳定可靠的数据传输方案，适配主流云平台。

通信接口 RS232 / RS485 / TTL 可选
网络制式 全网通 4G LTE Cat.1
支持频段 FDD/TDD-LTE, WCDMA, GSM
工作电压 DC 5V ~ 36V（宽压）
传输速率 上行 5Mbps / 下行 10Mbps
产品尺寸 85 × 55 × 25 mm
工作温度 -40℃ ~ +85℃
天线接口 SMA 外置天线
4G DTU 全网通

概述特点

产品介绍

4G-DTU 是一款工业级 4G LTE 数据传输终端，支持全网通（移动 / 联通 / 电信），可将串口设备通过 4G 网络接入互联网。产品支持 RS232、RS485、TTL 三种串口电平，适配 PLC、仪表、传感器等各类工业设备。

核心特点

  • 全网通 4G — 移动 / 联通 / 电信三网通用，Cat.1 稳定可靠
  • 多接口兼容 — RS232 / RS485 / TTL 三种串口可选，适配各类工业设备
  • 宽压供电 — DC 5V ~ 36V 宽电压，适应车载、太阳能、工业柜等场景
  • 云平台对接 — TCP / UDP / MQTT / HTTP 协议，支持阿里云、OneNET、EMQX 等
  • 不掉线 — 内置硬件看门狗 + 心跳包 + 注册包机制，7×24 小时稳定在线
  • 双模配置 — 串口 AT 指令 + 远程配置双模式，部署后仍可远程管理
  • 协议转换 — 支持 Modbus RTU ↔ Modbus TCP 协议转换
  • 导轨安装 — 导轨式外壳设计，便于工业机柜 DIN 导轨安装

典型应用

行业 应用场景
电力监控 远程抄表、配电柜监测、变压器状态上报
水利环保 供水管网压力/流量监测、水质/大气数据上报
充电桩 充电桩运营数据远程传输、状态监控
工业控制 PLC 远程编程与调试、设备预测性维护

快速入门

  1. 安装 SIM 卡 — 支持标准物联网卡，插入卡槽
  2. 连接天线 — 拧紧 SMA 4G 天线，确保信号良好
  3. 接入电源 — 推荐 DC 12V / 1A，PWR 指示灯亮起
  4. 等待联网 — NET 灯常亮表示已注册 4G 网络
  5. 配置服务器 — 串口发送 AT+SERVER=domain,port 设置云平台地址
  6. 进入透传 — AT+MODE=1 进入透传模式，数据自动上报云端

提示：详细 AT 指令说明请查看「相关下载」中的 AT 指令集手册。

引脚定义

# 引脚名称 功能说明 备注
1 VIN+ 电源正极，DC 5V ~ 36V 宽压输入 推荐 12V / 1A，建议加 TVS 防护
2 GND 电源地 与设备共地
3 485-A RS485 差分信号 A 端 接从设备 A 端，建议 120Ω 终端电阻
4 485-B RS485 差分信号 B 端 接从设备 B 端，建议 120Ω 终端电阻
5 TXD TTL 串口发送，接用户 MCU RX TTL 3.3V 电平
6 RXD TTL 串口接收，接用户 MCU TX TTL 3.3V 电平
7 LINK 4G 网络连接状态指示 联网成功输出高电平，可接 LED
8 RST 硬件复位，低电平有效 内部上拉，悬空即可
