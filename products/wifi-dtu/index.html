概述特点

产品介绍

WiFi-DTU 是一款工业级 WiFi 转串口数据透传模块，基于 ESP8266 芯片打造。模块可将用户设备通过串口接入 WiFi 网络，实现数据的双向透明传输，广泛应用于物联网数据采集与远程控制场景。

核心特点

  • 多协议支持 — TCP / UDP / MQTT / HTTP，适配各类云平台
  • 工业级设计 — 宽温 -40℃ ~ +85℃，适应恶劣现场环境
  • 超低功耗 — 休眠模式下功耗低至 50μA，适合电池供电场景
  • 简单易用 — AT 指令集配置，无需编程基础即可快速集成
  • 稳定可靠 — 硬件看门狗设计，7×24 小时运行不宕机
  • 灵活组网 — 支持 STA（连接路由器）和 AP（自建热点）双模式
  • Web 配置 — 内置 Web 管理页面，手机/电脑浏览器即可配置

典型应用

行业 应用场景
工业物联网 设备数据采集、远程监控、PLC 联网
智能家居 灯光控制、环境监测、家电联网
智慧农业 大棚温湿度监测、灌溉远程控制
能源管理 充电桩数据上报、电表远程抄表

快速入门

  1. 硬件连接：VCC → 3.3V~5V，GND → 地，TXD → MCU RX，RXD → MCU TX
  2. 上电：PWR 指示灯亮起，模块进入 STA 模式
  3. 连接 WiFi：发送 AT+WIFI=ssid,password 连接路由器
  4. 配置服务器：AT+TCP=192.168.1.100,8080 设置目标服务器
  5. 进入透传：AT+MODE=1 进入透传模式，串口数据直达云端

提示：详细 AT 指令说明请查看「相关下载」中的 AT 指令集手册。

引脚定义

# 引脚名称 功能说明 备注
1 VCC 电源正极，DC 3.3V ~ 5V 输入 建议并联 100μF 去耦电容
2 GND 电源地 与 MCU 共地
3 TXD 串口发送，接用户 MCU RX TTL 3.3V 电平
4 RXD 串口接收，接用户 MCU TX TTL 3.3V 电平
5 RST 硬件复位，低电平有效 内部上拉，悬空即可
6 LINK WiFi 连接状态指示 联网成功输出高电平
7 STA/AP 工作模式选择 悬空 = STA，接 GND = AP
8 NC 保留引脚 悬空
